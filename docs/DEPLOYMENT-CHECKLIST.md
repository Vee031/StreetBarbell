# Production deployment checklist

- [ ] Confirm GitHub ZIP/repository contains no `.env.local`
- [ ] Run `npm run verify`
- [ ] Change the temporary distributor access code
- [ ] Add all three environment variables to Vercel Preview and Production
- [ ] Connect the `main` branch
- [ ] Deploy and test `/en`, `/cs`, products, gallery and contact
- [ ] Unlock the configurator and generate one EN and one CZ recommendation
- [ ] Download and open one PDF summary
- [ ] Test mobile navigation
- [ ] Add `streetbarbell.cz` and `www.streetbarbell.cz`
- [ ] Verify canonical redirect between apex and `www`
- [ ] Test email and WhatsApp buttons
- [ ] Replace temporary/remote imagery before final marketing launch
